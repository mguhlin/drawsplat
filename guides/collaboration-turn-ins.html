Feature Guide

Collaboration and Turn-Ins

Set up shared rooms, student links, assignment layers, and turn-ins when DrawSplatTM is connected to the optional Google Apps Script backend.

Key ideas

  • Teacher mode: full control of panels, backgrounds, templates, and setup.
  • Student mode: student-facing links hide provider settings and restrict teacher-only controls.
  • Assignment mode: protects teacher-layer objects while students add work on the student layer.
  • Cloud rooms: use a shared room name and optional password to sync board state.
  • Turn-ins: students can submit board work for teacher review when configured.

Teacher workflow

  1. Complete Google Drive and Sheets setup.
  2. Open Teacher Admin and save the Apps Script Web App URL.
  3. Open the board, switch to Education Tools, and enable assignment mode.
  4. Create the panels, prompts, backgrounds, and teacher-layer objects students need.
  5. Start cloud sync with a unique room name and optional room password.
  6. Copy the student link from Teacher Admin or the board and share it with students.

Student workflow

  1. Open the student link.
  2. Enter the room password if one is required.
  3. Add notes, drawings, text, images, diagrams, or responses on the student layer.
  4. Use turn-in when the teacher asks for a submitted copy.

Tips

  • Use unique room names for each class period or activity.
  • Use the narrowest Google Apps Script deployment access setting that works for your school.
  • Keep teacher prompts on the teacher layer before students join.
  • Use comments and the moderation dashboard to review feedback pins and submissions.
