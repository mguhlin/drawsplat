Feature Guide

Classroom Widgets and Standalone Tools

Use built-in widgets on the board and open standalone classroom tools in new tabs for routines, games, generators, and quick activities.

Board widgets

  • Poll: collect quick visible responses.
  • Random Pick: choose names, groups, or topics.
  • Wheel Spinner: spin options for prompts or teams.
  • Timer: show work time directly on the board.
  • Traffic Light and Work Mode: display classroom status.
  • Scoreboard and Progress Race: track review games or group progress.
  • Big Link: post a large readable URL or resource link.

Standalone tools

  • Coin Flipper, Dice Roller, Dicebreaker Creator, Story Wheel, Word Search Maker, Meme Puzzle, Rubric Builder, Markdown Studio, Concept Map Studio, Animated GIF Maker, and Big Link open in separate tabs.
  • Use them beside DrawSplatTM when you need a focused activity generator rather than an object on the board.
  • Most standalone tools are self-contained and do not require an account.

Animated GIF Maker

  • What it does. Stitch a sequence of PNG / JPG / WebP / GIF still images into a single animated GIF with per-frame delay, loop count, and optional max-width scaling.
  • Small files. Choose Best color quality and pick a palette size — 256 for photos, 128 or 64 for typical classroom diagrams. Most maps under 100 KB at 64 colors.
  • Dithering. Optional Floyd-Steinberg dithering smooths gradient banding (slightly larger files; turn off for flat-color content).
  • Video export. Click Export as Video for MP4 (Chrome / Edge 94+, Firefox 130+, Safari 16.4+) or WebM (older browsers) when you need the animation in a tool that doesn't accept GIF — PowerPoint, Google Slides, YouTube, or an LMS file uploader.
  • Privacy. Frames stay on the student's / teacher's device; nothing is uploaded. The encoder runs in the browser.

Concept Map Studio

  • Three-tier hierarchy: mark each node level 1 / 2 / 3 to scale box and text size automatically.
  • Outline view under the canvas shows the map as a traditional Roman/Letter/Number academic outline. Copy as Markdown or hand off to Markdown Studio for further editing.
  • Embed in a website: click Copy Embed Code to grab an <iframe> snippet for Google Sites or WordPress — viewers get pan, zoom, and clickable node links.
  • AI prompt-ready: download the format spec (v2) link and hand it to ChatGPT / Claude / Gemini to generate JSON you can load directly.

Suggested routines

  • Start class with a timer and a Big Link to the task.
  • Use Random Pick for discussion order.
  • Use Markdown Studio to clean student or teacher Markdown before publishing — then jump to Concept Map Studio for a visual outline.
  • Use Animated GIF Maker to turn a sequence of student screenshots into a quick "how-to" loop or to show stages of a process.
  • Use Rubric Builder before a project work session, then paste or export the rubric.
  • Use Word Search Maker, Story Wheel, or Dicebreaker Creator as quick warm-up tools.
