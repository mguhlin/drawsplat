About DrawSplatTM

Version v3.0

DrawSplatTM is a free K-16 interactive whiteboard for classrooms and small-group collaboration. Draw, place sticky notes, attach audio, organize panels, and share boards via Google Drive.

Vibe-coded by Miguel Guhlin.

Open source under AGPL-3.0-or-later, with separate commercial licenses available for school, district, hosted, or closed-source deployments.

Teacher Admin

Google setup, room defaults, and classroom sharing settings are managed on the Teacher Admin page so students never see provider settings.

Reset

Deletes every panel and starts with a blank single panel. This can't be undone — save the board first if you need a copy.

Google Drive + Sheets Setup

  1. Create a Google Sheet named DrawSplatTM Saves.
  2. Open Extensions → Apps Script.
  3. Paste the included apps-script/Code.gs code.
  4. Deploy as Web app, run as yourself, accessible to your users.
  5. Copy the Web App URL into Script URL.

The script stores board JSON and PNG exports in Drive and logs saves in the Sheet. It can also store shared room states for cloud collaboration.
