# DrawSplat Coloring Book Assets

These coloring pages are original DrawSplat line-art assets for classroom coloring. The app supports PNG, JPG, and JPEG files for each listed page.

- Count: 80 pages
- Categories: plants, insects, dinosaurs, animals
- License: CC0 1.0 public-domain dedication for unrestricted classroom and project use
- Preferred filenames: use the manifest IDs with `.png`, `.jpg`, or `.jpeg` extensions, such as `plants-sunflower.jpg`

The artwork is original line art. It is not copied, traced, or adapted from Crayola or other third-party coloring pages.
