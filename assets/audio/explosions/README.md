# Red Library Explosions

Public-domain explosion samples used by Castles & Catapults (per-munition
impact sounds) and the whiteboard TNT detonation.

- **Source:** Red Library Explosions sound pack, mirrored on the Internet
  Archive at <https://archive.org/details/Red_Library_Explosions>.
- **License:** CC0 1.0 Universal — public domain dedication.
  <https://creativecommons.org/publicdomain/zero/1.0/>
- No attribution required, but credited here for provenance.

The files are referenced by slugified filename from JavaScript. If you add
or rename anything, update the lookup arrays in:

- `games/castles/index.html` — per-munition impact pools and TNT-style picks.
- `assets/js/app.js` — `playTntBoom` in the whiteboard.

Keep new additions CC0 (or another compatible license) so the project stays
AGPL-3.0-or-later-clean.
