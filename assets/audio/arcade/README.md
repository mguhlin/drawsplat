# Analog Samples — Arcade Sound Effects

Numbered arcade-style SFX (clicks, pings, lasers, hums, jingles, zaps,
buzzes) for use across the DrawSplat games suite.

- **Source pack:** "Analog Samples — Arcade Sound Effects", 53 clips.

  > Samples, licensed under CC BY 4.0.

  Surface this credit somewhere users can find it. The pages/about.html
  acknowledgements section is the canonical home; the per-game README in
  this folder is a secondary record.

Files are referenced from JS as `arcade-NN.mp3` (zero-padded). See
`placement-map.md` for proposed game-by-game pairings.
