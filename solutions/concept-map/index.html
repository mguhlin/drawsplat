Mode

Shift-click nodes to add to the selection. Drag empty canvas to rubber-band select. Double-click empty canvas to add a node where you clicked. Middle-mouse drag or Pan toggle to pan the view. Wheel / Ctrl+wheel to zoom.

Edit selected

Click a node or connector to select. Shift-click to add another. Drag empty canvas to rubber-band select. Click a connector line to select it — a "+ label" pill appears so you can add or edit its label.

Concept maps stay in your browser only — nothing is uploaded. Use Save .json to keep your work as a local file, then Load .json to re-open it later. Ctrl/Cmd-click a node with a link to open it in a new tab.
