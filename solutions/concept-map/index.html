Layout

Syntax

One node per line. Use leading hyphens to nest:
Root
- Child
-- Grandchild

Each row supports up to three pipe-separated fields:
label | link | emoji-or-image

The image field accepts an emoji (e.g. 🌱) or an attached image (use the Add image button below the editor).

About

Concept maps are rendered locally in your browser as SVG. Nothing is uploaded. Pair with the DrawSplatTM whiteboard to drop a finished map onto a board.
